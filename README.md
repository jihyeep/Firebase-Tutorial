Firebase를 사용한 앱 프로젝트
==========
> SwiftUI

멋쟁이사자처럼 iOS-AppSchool 5기 강의 기반 실습

## **목차**

1. **NoteApp**:
   - Firebase 설정
   - Firestore 설정: 데이터 CRUD, 사용자 정보 저장
   - Authentication: SignIn, SignUp, SignOut, ResetPassword 기능
2. **Socially**:
   - @FirestoreQuery 활용
   - Apple 로그인 연동
   - Storage를 사용한 이미지 파일 관리
   - Analytics를 사용한 사용자 로그 관리
